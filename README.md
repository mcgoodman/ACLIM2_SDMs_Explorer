## ACLIM2 SDMs Web App

This repository contains code to build and deploy a web application to browse species distribu
tion and predator-prey overlap from [Goodman et al. (2025)](https://doi.org/10.1111/faf.12875). Source code for the analyses can be found at [mcgoodman/EBS_range_projections](https://github.com/mcgoodman/EBS_range_projections/tree/main). The app is hosted on [shinyapps.io](https://mgoodman.shinyapps.io/aclim2_sdms_explorer/) and the [NOAA NMFS server](https://connect.fisheries.noaa.gov/aclim2_sdms_explorer/).
